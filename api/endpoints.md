# API Endpoints
## Users
```http
GET api/v1/users/
```
**Description** : Returns a list of users

**Auth required** : YES

**Permissions required** : None

### Parameters
|  Name | Required |            Description            | Default | Example |
|:------|:--------:|:----------------------------------|:-------:|:-------:|
| group | No       | Filter by groupID                 | None    | `1`     |
| year  | No       | Filter by graduation year         | None    | `2021`  |
